CyberDoc
Интеллектуальная платформа для анализа безопасности веб-проектов и обучения защищенной разработке.

Технологический стек
Runtime: Node.js

Framework: Express

AI Engine: Langflow (LLM)

Protocols: TLS/SSL, TCP/IP

Функционал
Автоматизированное сканирование сетевых портов.

Проверка валидности и конфигурации SSL-сертификатов.

Анализ исходного кода на наличие уязвимостей (XSS, Open Redirect).

Генерация исправлений и рекомендаций с помощью ИИ.

Установка и запуск
Клонирование репозитория

Bash

git clone https://github.com/TeemUr-Check/CyberDoc.git
cd CyberDoc
Установка зависимостей

Bash

npm install
Запуск сервера

Bash

node server.js
Приложение запускается на локальном хосте. Все API-ключи встроены в систему.

Особенности архитектуры
Асинхронность: одновременная проверка нескольких параметров безопасности.

Интерактивность: предоставление обратной связи в формате чата.

Точность: распознавание и устранение до 90% типовых ошибок безопасности.
